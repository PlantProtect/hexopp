集线器、交换机和路由器的区别

Posted on 2023-02-26 In 计算机网络

集线器、交换机和路由器的区别（The differences bettwen Switch Router and Hub）

集线器（hub），当一台电脑发送给另一台电脑数据包时，所有端口上连接的电脑都能收到数据包，同时会造成网络的拥堵，极大浪费了带宽。

hub that every computer connectd can received the data packet.

交换机(switch)，与集线器类似，但更智能。可以解析每一台设备的ip地址（称为MAC 地址），只向特定端口的设备发送数据包。


路由器，连接不同网络，可以理解成不同网络之间的大门（gateway）

A router is a device that routes or fowards data from one network to another based on their IP address. When a data packet is received from the router,the router inspects the data’s IP address and determines if the packet was meant for its own network or if its meant for another network.

If the routerdetermines that the data packet is meant for its own network,it receives it. But if it’s not meant for it’s own network, it sends it off to another network.
